## Aseprite Palette Pack

These palette files are in `.gpl` format and can be imported directly in Aseprite.

### Import in Aseprite

1. Open Aseprite.
2. Open the palette panel menu.
3. Choose `Load Palette...`.
4. Select one of the `.gpl` files in this folder.

### Included Palette Types

- `lu-card-art.gpl`
- `lu-enemy.gpl`
- `hp-card-art.gpl`
- `hp-enemy.gpl`
- `j-card-art.gpl`
- `j-enemy.gpl`
- `neutral-card-art.gpl`
- `neutral-enemy.gpl`
- `neutral-ui.gpl`

### Notes

- `#C8262D` is included as a shared blood/injury accent color.
- Card-art palettes include neutral support colors as allowed by project rules.
- Enemy palettes include faction colors plus neutral foreground support colors.
